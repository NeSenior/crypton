# Crypton

Crypton — game engine library based on OpenGL.

## About

Crypton is a C++ library designed to simplify 2D game development.
It provides rendering, basic engine structure and low-level control over graphics.

## Installation

Add the library to your project:

- Place the `.a` file into your libraries directory
- Include the `.h` headers in your project

Or compile from source and link it manually.

## Language Support

Currently available for C++.
Ports for other languages are planned.
